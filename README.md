## 🔧 설치 방법

아래 명령어를 실행하여 프로젝트를 설치할 수 있습니다.

```bash
# 프로젝트 클론
git clone https://github.com/awsdevgen/LGUplus-B2B-AX-dev.git
cd LGUplus-B2B-AX-dev

# 필수 패키지 설치
pip install -r requirements.txt

# 실행
python main.py
```

## 💡 기능 및 특징
✅ STT (Speech-to-Text) 변환

✅ S3 연동 가능 (AWS와 쉽게 연결 가능)
